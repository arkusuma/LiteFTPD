LiteFTPD 0.3
============

About
-----
LiteFTPD is a small multi-thread FTP server for Windows platform.
When running, LiteFTPD will share all of your drive to public.

Warning:  LiteFTPD is not suited for use over Internet since it will
          make your data exposed to public.

Unique Features
---------------
  o Remote desktop sharing, see LiteClient.exe
